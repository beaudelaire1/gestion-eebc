# Captures d'écran - Guide Utilisateur

Ce dossier contient les captures d'écran utilisées dans le guide utilisateur de l'application Gestion EEBC.

## Liste des captures

| Fichier | Description | Module |
|---------|-------------|--------|
| `01-login-page.png` | Page de connexion de l'application | Authentification |
| `02-dashboard.png` | Tableau de bord principal avec statistiques | Accueil |
| `03-membres-list.png` | Liste des membres avec filtres et recherche | Gestion des Membres |
| `04-club-biblique.png` | Vue d'ensemble du club biblique | Club Biblique |
| `05-cultes.png` | Planning des cultes | Cultes/Worship |
| `06-finance.png` | Tableau de bord financier avec graphiques | Finance/Trésorerie |
| `07-calendar.png` | Calendrier intelligent des événements | Événements |
| `08-communication.png` | Module des annonces | Communication |

## Informations techniques

- **Format** : PNG
- **Résolution** : Variable (adaptée à la taille de la page)
- **Date de capture** : Janvier 2026
- **Version de l'application** : 1.1

## Mise à jour

Pour mettre à jour les captures d'écran :

1. Lancer l'application en mode développement
2. Se connecter avec un compte administrateur
3. Naviguer vers chaque module
4. Prendre une capture d'écran complète de la page
5. Nommer les fichiers selon la convention : `XX-nom-module.png`
6. Optimiser les images si nécessaire (compression PNG)

## Notes

- Les captures sont prises sur un écran en mode clair
- Les données affichées sont des données de test/exemple
- Les captures montrent l'interface sans données réelles pour respecter la confidentialité
